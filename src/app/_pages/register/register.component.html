Cadastro de Digital

Para usar bem um sistema biométrico é necessário guardar dados. No caso de um sensor de digitais, é preciso que dados de impressões digitais possam ser armazenadas de forma rápida e eficiente. Sabendo disso, veremos nesta aba o quão fácil e prático é realizar o cadastro de uma digital no sensor AS608 conectado a um Arduino.

Conexão e configuração do Arduino

Primeiramente, faça a seguinte conexão:

No menu Arquivo escolha Exemplos > Adafruit Figerprint Sensor Library > Enroll, como ilustra a figura abaixo:

Após encontrar o leitor biométrico, cadastre o dedo digitando um número de 1 a 127 e clique em enviar. Observe que é solicitado por o dedo duas vezes.

É preciso aguardar até que apareça o seguinte:

Você será solicitado a colocar o mesmo dedo duas vezes no scanner. Se você receber a mensagem *"Impressões correspondentes!"* (Prints matched!), conforme mostrado abaixo, significa que sua impressão digital foi armazenada com sucesso. Se não, repita o processo até ter sucesso.
